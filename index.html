Links and Images

We're going to add a link now. You can do so with an anchor element "a".

The anchor element needs an href attribute with the url.

Add a target attribute with value _blank if you want a new tab. Default is _self.

For security reasons, add a rel attribute with value "noopener noreferrer". Prevents opener and referrer knowledge transmission.
